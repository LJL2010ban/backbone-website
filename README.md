## generate-sapling-pc

音悦台PC端项目脚手架，项目所使用的工具如下：

- webpack
- gulp
- npm scripts
- bower
- browser-sync

## How use it?

download project

	git clone https://github.com/sapling-team/generator-sapling-pc.git

install global gulp and bower

    npm install -g bower
    
    npm install -g gulp

Node version v4.4.1
